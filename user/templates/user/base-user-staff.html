Introducción

Desde Knives Ecommerce Simulation, le proporcionamos la posibilidad de crear un usuario con rango de "Staff" para poder brindarle una mayor experiencia y descubra las capacidades de GdeG como desarrollador.

Para poder preservar la experiencia de otros usuarios agregamos algunas restricciones. La sesión del usuario con rango de "Staff" dura 1 hs, desde la selección del rango, o cuando se cierre sesión de la cuenta. Pasado el tiempo se le inhabilitara el usuario y se eliminaran todos los cambios que se hayan realizado el mismo, con el proposito de preservar la experencia de otros usuarios manteniendo la bases del proyecto.

Uso y Comportamiento

El uso de las funciones de "Staff" es muy intuitivo. Únicamente para los usuarios con este rol, aparecerán botones en las correspondientes secciones de la aplicación web, además le brindamos un "Staff Dashbord" donde podrán acceder a la información y funcionalidades en un solo lugar. Al Dashbord acceden desde la barra de navegación, apretando el botón cuál tiene el nombre de usuario, el cual desplegara un menú en el cual tendrá que cliquear la opción de "Staff Dashbord".
